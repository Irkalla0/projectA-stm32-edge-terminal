# 项目A简历条目（可直接用）

项目名：`项目A：分布式多传感边缘监测终端（STM32F407 + FreeRTOS + MQTT）`

## 三条投递版（推荐）
- 负责 STM32 多源采集与告警链路开发，完成设备端到上位机闭环，单轮演示 `763/763` 帧 CRC 全通过。
- 设计并落地 UART 升级协议与 Boot State 回滚策略，实现升级状态可观测、失败可回退，升级闭环支持 `UPG_BEGIN/UPG_DATA/UPG_END/UPG_ACTIVATE/UPG_CONFIRM`。
- 在原系统上扩展 MQTT 多节点汇聚、异常检测与短期预测能力，形成“采集-分析-预测-告警”一体化方案并输出可复现实验报告。

## 面试扩展点（答问版本）
- 系统架构：设备侧（STM32 + 传感）/通信侧（UART + MQTT）/平台侧（聚合+分析+存储）。
- 关键难点：升级长报文抗干扰、双串口命令解析冲突、BootState持久化一致性。
- 可靠性措施：重试与超时机制、状态机约束、升级日志证据链、异常场景回放。
